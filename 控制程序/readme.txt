实验器材:
	探索者STM32F4开发板
	
实验目的:
	实现驱动smart weeding bot的
硬件资源:
	1,串口1(波特率:115200,PA9/PA10连接在板载USB转串口芯片CH340上面)
	
实验现象:
	本实验下载后，串口1将不停的输出t的值，打开电脑串口调试助手即可查看现象。 
智能精准化识别、定点定量除草机构通过多个旋转电机、连接同步带带,可实现三个喷头同时在±90°的范围内任意转动,通过丝杠,可实现在固定点不变的情况下喷头的位置微调。利用不同PWM波形的调控，可以对喷头打大小进行无极切换。通过液滴粒子动力学建模、液滴理论沉积覆盖区域研究，得到农药喷头对单株靶向施药的角度参数。
